# Assets

This folder contains non-code assets used by S.T.A.L.K.E.R. Mod Launcher.

## Layout

- `app/` — application icon used by the Windows executable.
- `icons/mod-actions/` — icons for mod list and mod actions.
- `icons/profile-actions/` — icons for profile creation, import, export, copy and delete actions.
- `icons/navigation/` — arrows and refresh/navigation icons.
- `icons/profile-icons/` — optional profile icon artwork.
- `screenshots/` — README screenshots.
- `sounds/pda/` — UI sounds.

## Notes

- PDA sounds are from the ModDB package `PDA Sounds by SAM (3 in 1)`, marked Public Domain on ModDB.
- UI icons were collected through Iconify/Figma and keep their original upstream icon-set licenses.
- Game names, references and visual style belong to their respective rights holders. The launcher is an independent community tool.
