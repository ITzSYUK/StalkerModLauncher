# S.T.A.L.K.E.R. Mod Launcher

Windows-лаунчер для раздельного запуска локальных модификаций S.T.A.L.K.E.R.

Он помогает собрать профиль из базовой игры и папок модов, запустить его в отдельной рабочей среде и не менять исходные файлы игры. Подходит для локальных DRM-free установок **Shadow of Chernobyl**, **Clear Sky**, **Call of Pripyat** и автономных сборок на базе X-Ray, включая Anomaly и проекты на OGSR, если их структура распознаётся лаунчером.

Создаваемые профили не копируют игру целиком: лаунчер подключает исходные файлы ссылками, поэтому обычно занимает минимум дополнительного места на диске.

## Что умеет

- создавать обычные и автономные профили;
- подключать несколько модов, включать их и менять порядок перетаскиванием;
- автоматически находить EXE, но позволять выбрать его вручную;
- хранить сохранения, настройки и логи обычного профиля отдельно от игры;
- показывать состояние профиля, последний лог, crash dump и сведения о workspace;
- импортировать, экспортировать и копировать профили;
- вести учёт времени в игре, показывать необязательный статус Discord и открывать скриншоты;
- открывать встроенный браузер модификаций AP-PRO для ТЧ, ЧН и ЗП.

## Быстрый старт

1. Нажмите **Создать** и задайте имя профиля.
2. Выберите тип:
   - **Обычный профиль**: укажите папку установленной игры и добавьте папки модов.
   - **Автономный мод**: укажите одну корневую папку готовой сборки.
3. Проверьте найденный EXE в **Настройках**. При необходимости выберите другой вручную.
4. Запустите профиль кнопкой **Запустить**.

Моды ниже в списке имеют больший приоритет: именно так обычно ставится патч поверх основного мода.

## Браузер модификаций

Кнопка **Браузер модификаций** находится в панели профилей. Она открывает каталог материалов [AP-PRO.RU](https://ap-pro.ru/) с карточками модов для Тени Чернобыля, Чистого Неба и Зова Припяти. Новые страницы каталога подгружаются при прокрутке списка; нажатие на карточку открывает исходную страницу мода в браузере. Лаунчер не скачивает и не устанавливает моды из каталога.

## Требования

- Windows 10 или Windows 11 x64;
- установленный .NET 8 Desktop Runtime x64 для готовой сборки;
- .NET 8 SDK для сборки из исходного кода.

## Сборка

```powershell
dotnet build .\StalkerModLauncher.sln
dotnet test .\StalkerModLauncher.sln -c Release
dotnet run --project .\src\StalkerModLauncher\StalkerModLauncher.csproj
```
