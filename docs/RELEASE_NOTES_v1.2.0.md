# S.T.A.L.K.E.R. Mod Launcher v1.2.0

`v1.2.0` развивает профильную модель лаунчера и добавляет экспериментальный режим USVFS, не заменяя проверенный Workspace.

## Основные изменения

- Два backend-а запуска для обычных профилей:
  - **Workspace — стабильный** остаётся режимом по умолчанию;
  - **USVFS — экспериментальный** использует официальный runtime Mod Organizer 2.
- Поддержка USVFS для x64- и x86-движков через отдельный 32-битный helper.
- Единые `FileLayerPlan`, `OverlayManifest` и `LaunchPlan` для выбора итоговых файлов, EXE и порядка модов.
- Выбор renderer для Anomaly в USVFS: Auto, DX8, DX9, DX10, DX11 и AVX.
- Надёжная привязка workspace к ID профиля: переименование больше не создаёт новую рабочую папку.
- Безопасное удаление профилей с проверкой marker-файла.
- Изоляция `fsgame.ltx`, `user.ltx`, сохранений, логов, скриншотов и writable-конфигураций.
- Улучшенные preflight-проверки, журналирование и диагностика источника запускаемого EXE.
- Более удобный мастер профиля и увеличенное окно выбора найденных модов.
- Браузер модификаций AP-PRO с категориями, поиском, последовательной подгрузкой страниц и отменой устаревших запросов.
- Обновлённые README, техническая документация и сведения о сторонних компонентах.

## Важно

USVFS в этом релизе предназначен для осознанного тестирования. Совместимость зависит от конкретного движка и способа запуска. Если сборка работает нестабильно, выберите **Workspace — стабильный** в настройках профиля.

Для USVFS требуется Microsoft Visual C++ 2015-2022 Redistributable соответствующей архитектуры. Framework-dependent версия также требует .NET 8 Desktop Runtime x64; standalone-версия содержит .NET внутри.

## Известные ограничения

- Отдельные wrappers и нестандартные X-Ray-сборки могут работать только через Workspace.
- Symbolic link между разными дисками может потребовать Режим разработчика Windows или права администратора.
- Пути игры и модов абсолютные; после переноса их нужно обновить в профиле.
- Браузер модификаций зависит от доступности и текущей разметки AP-PRO.
